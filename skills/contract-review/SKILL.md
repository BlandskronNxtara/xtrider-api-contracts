# Contract Review

1. Lee `AGENTS.md`, `CONTRACT.md` y `VERSIONING.md`.
2. Identifica productor, consumidores y version afectada.
3. Busca eliminaciones, renombres, cambios de tipo o campos obligatorios nuevos.
4. Verifica ejemplos y `CHANGELOG.md`.
5. Ejecuta scripts de validacion.
