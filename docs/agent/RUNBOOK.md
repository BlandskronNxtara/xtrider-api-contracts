# Agent Runbook

## Antes de editar

1. Leer `AGENTS.md`.
2. Leer `CONTRACT.md`.
3. Identificar servicio productor y consumidores.
4. Revisar `VERSIONING.md`.
5. Buscar contratos existentes con `rg`.

## Despues de editar

1. Agregar o actualizar ejemplos.
2. Actualizar `CHANGELOG.md`.
3. Ejecutar validaciones.
4. Reportar servicios impactados y breaking changes.
