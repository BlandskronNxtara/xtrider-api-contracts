# Overview

`xtrider-api-contracts` separa contratos publicos de implementaciones internas.

## Responsabilidades

- Definir formatos compartidos de respuesta, error, eventos, jobs, webhooks e IA.
- Documentar compatibilidad y versionado.
- Dar contexto operativo a agentes IA.
- Preparar generacion futura de SDKs sin acoplar servicios.

## No responsabilidades

- No ejecutar negocio.
- No definir modelos internos.
- No conectar bases de datos.
- No publicar secretos.
- No reemplazar la documentacion especifica de cada servicio.
